### 本プロジェクトについて
本プロジェクトは、**NIST Cybersecurity Framework（CSF）2.0** の構造および考え方を参考に、  
個人開発者が作成した **非公式のセルフアセスメントツール**です。

本ツールは、組織のセキュリティ対策状況を**簡易的・俯瞰的に把握すること**を目的としており、  
以下を網羅的に反映するものではありません。

- 個別組織の詳細な状況  
- 業種・業界固有の要件  
- 最新の脅威動向  
- 法令、規制、契約上の要件  

本プロジェクトは **NIST による認定、承認、推奨、保証を受けたものではありません**。
